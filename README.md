# MinedeathPanel — DEPRECATED

This repo held the Twitch Extension panel (Bits / channel-points actions).
That feature was dropped from the product; the suite now lives in the unified
[**minedeath**](https://github.com/TardHelmet/minedeath) monorepo, and viewer
interaction happens through the web store at
[minedeath.store](https://minedeath.store) and the Twitch chat bot.

If the extension is still registered in the Twitch developer console, it can
be deleted. Archive this repo.
